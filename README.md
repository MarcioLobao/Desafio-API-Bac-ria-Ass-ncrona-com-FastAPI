Neste desafio voce irá projetar e implementar uma API RESFull assíncrona  usando FastAPI para gerenciar operações bancárias de depósitos e saques  vinculadas a contas correntes. Este desafio irá lhe proporcionar a experiêcia de construir uma aplicação backend moderna e eficiente que utiliza autenticação JWT e praticas recomendadas de design de APIs
